## About

A minimal developer portfolio focused on content over decoration. No frameworks, no unnecessary dependencies — just a clean, fast site that lets the work speak for itself.

- **Minimal by design** — dark monospace aesthetic, zero visual noise
- **Fast** — static generation, no client-side bloat
- **Fully responsive** — works on every screen size
- **CV viewer** — built-in PDF viewer so employers can read without downloading

## Features

| | |
|---|---|
| **Projects page** | Curated list of projects with tech stack, links, and dates |
| **CV viewer** | Inline PDF viewer with download option |
| **Email copy** | One-click email copy with visual feedback |
| **Fade-in animations** | Staggered entrance animations on all pages |
| **Monospace typography** | Geist Mono throughout for a consistent developer aesthetic |
| **Mobile responsive** | Fully usable on all devices |

## Getting Started

```bash
# Clone the repository
git clone https://github.com/milnee/Portfolio.git
cd Portfolio

# Install dependencies
bun install

# Start dev server
bun dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
